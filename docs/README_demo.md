# Docsify 编辑器演示

欢迎使用 **Docsify 编辑器**！

## ✨ 快速开始

点击右上角的 **✏️ Edit** 按钮，或按 `Ctrl+E` 进入编辑模式。

## 📝 功能特性

### 1. 实时预览

在编辑模式下，你可以：
- 左侧编辑 Markdown
- 右侧实时查看渲染效果
- 所见即所得

### 2. 导出功能

编辑完成后，你可以：
- 📋 复制到剪贴板
- 💾 下载为 .md 文件

### 3. 支持富文本

编辑器支持所有 Markdown 语法：

**粗体文本** 和 *斜体文本*

> 引用块
> 可以多行

```python
# 代码块示例
def hello_world():
    print("Hello, Docsify Editor!")
```

### 4. 数学公式

行内公式：$E = mc^2$

块级公式：

$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$

### 5. 表格支持

| 功能 | 状态 | 说明 |
|------|------|------|
| 编辑 | ✅ | 支持实时编辑 |
| 预览 | ✅ | 实时渲染预览 |
| 导出 | ✅ | 复制/下载 |
| 保存 | ❌ | 不自动保存 |

### 6. 列表

有序列表：
1. 进入编辑模式
2. 修改内容
3. 导出文件

无序列表：
- 简单易用
- 功能强大
- 纯前端实现

## 🎯 使用场景

### 场景一：快速修正

发现文档中的错误时，可以：
1. 点击 Edit 进入编辑
2. 修正错误
3. 复制修改后的内容
4. 提交给维护者

### 场景二：添加批注

阅读时想要添加笔记：
1. 进入编辑模式
2. 添加你的批注
3. 下载带批注的版本

### 场景三：格式转换

需要其他格式时：
1. 在编辑器中调整格式
2. 复制 Markdown
3. 粘贴到 Notion/语雀/Typora
4. 导出为你需要的格式

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+E` / `Cmd+E` | 切换编辑模式 |
| `Ctrl+S` / `Cmd+S` | 下载文件 |
| `ESC` | 退出编辑模式 |

## 💡 小提示

- ⚠️ 编辑器**不会自动保存**，记得导出！
- 📁 下载的文件名会加上 `_edited` 后缀
- 🔒 所有操作都在浏览器本地完成，数据不会上传

## 🚀 立即体验

现在就点击右上角的 **Edit** 按钮试试看！
